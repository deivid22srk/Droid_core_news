# Skill: Criar Notícias Droid_Core

Esta skill orienta o agente na criação de notícias técnicas seguindo o padrão do Droid_Core.

## Diretrizes de Conteúdo
1. **Título:** Chamativo, foco em tecnologia/emulação/mobile.
2. **Excertos:** Resumo de 2 linhas focado no impacto da notícia.
3. **Corpo da Notícia:** 
   - Linguagem direta, técnica e informativa.
   - Usar `### Header` para organizar seções.
   - **Obrigatório:** Listar todas as fontes de onde as informações foram extraídas ao final da notícia.

## Formatação Técnica
1. **Links Clicáveis:** Sempre usar o formato `[Título](URL)`.
2. **Comandos (Copyable):** Blocos de código devem usar triple backticks com a linguagem (ex: ````bash`) seguido de `CodeBlock` no componente.
   - Exemplo:
     ```bash
     comando_aqui
     ```
3. **Citações/IA:** Sempre incluir o aviso de revisão caso a notícia tenha sido estruturada com auxílio de IA.

## Checklist de Publicação
- [ ] Título claro.
- [ ] Excerpt impactante.
- [ ] Fonte dos dados linkada.
- [ ] Código/Comandos formatados para fácil cópia.
- [ ] Aviso de IA integrado ao fluxo (via UI do Droid_Core).

---
*Ao criar notícia, sempre cite a URL da fonte original de maneira explícita.*
